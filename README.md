# 🧠 Mastermind Web

A web-based implementation of the classic code-breaking game **Mastermind**, built with Ruby and Sinatra. This project was developed to practice object-oriented programming, clean code principles, and MVC architecture concepts as part of The Odin Project curriculum.

## 🎮 Play the Game

The goal of Mastermind is to guess the secret code randomly generated by the computer. You have 10 attempts to crack a code made of 4 colors.

### Rules & Feedback
After each guess, the computer provides feedback using small pins:
* ⬛ **Black Pin:** You guessed a correct color in the **correct position**.
* ⬜ **White Pin:** You guessed a correct color, but it's in the **wrong position**.
* **Empty:** The color is not part of the secret code.

*Note: The feedback pins do not necessarily correspond to the exact order of your guess. You must deduce which pin belongs to which color!*

## ✨ Features

* **Interactive Web UI:** A sleek, custom-styled dark mode interface.
* **Real-time Visual Feedback:** Colors on the board update instantly as you select them from the dropdown menus (powered by Vanilla JavaScript).
* **Robust Game Logic:** Accurately calculates feedback even when the secret code contains duplicate colors, preventing false positives.
* **Session Management:** Uses Sinatra sessions to keep track of the game state gracefully without a database.
* **Tooltips:** Hover over the feedback pins for quick rule reminders.

## 🛠️ Technologies Used

* **Backend:** Ruby
* **Framework:** Sinatra (Web routing and session management)
* **Autoloading:** Zeitwerk (Keeps the `lib/` directory clean and organized)
* **Frontend:** ERB (Embedded Ruby), HTML5, CSS3, Vanilla JavaScript

**2. Install dependencies**
Make sure you have ruby installed. Then, run: bundle install
**3. Start the server**
ruby app.rb
